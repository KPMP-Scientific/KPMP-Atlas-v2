# Chromatin Viewer for KPMP Atlas v2 multiome data

## Setup

```sh
pnpm install
```

## Development

```sh
pnpm run start
```

### Build

```sh
pnpm run build
```

### Deploy

```sh
pnpm run deploy
```

Will deploy to gh-pages branch, with published site at:

https://KPMP-Scientific.github.io/KPMP-Atlas-v2/chromatin-viewer/
